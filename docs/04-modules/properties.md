# Properties Module

## Objetivo

Administrar inmuebles comercializables.

---

## Operaciones soportadas

* Venta
* Alquiler
* Alquiler temporario

Una misma propiedad puede tener múltiples operaciones simultáneamente.

---

## Información física

* Tipo de inmueble
* Dirección
* Ubicación
* Metros totales
* Metros cubiertos
* Metros descubiertos
* Frente del terreno
* Fondo del terreno
* Ambientes
* Dormitorios
* Baños
* Toilettes
* Cocheras
* Antigüedad
* Orientación
* Disposición
* Luminosidad

---

## Información comercial

* Operación
* Estado
* Precio
* Moneda
* Expensas
* Destacado

---

## Imágenes

* Portada
* Galería
* Orden manual

---

## Características

### Generales

Ejemplos:

* Apto crédito
* Apto profesional
* Uso comercial

### Servicios

Ejemplos:

* Agua corriente
* Gas natural
* Cloacas
* Internet

### Ambientes

Ejemplos:

* Living
* Comedor
* Jardín
* Patio
* Lavadero

### Amenities

Ejemplos:

* Pileta
* Parrilla
* Quincho
* Seguridad
* Aire acondicionado

---

## Permisos

### AGENT

* Crear propiedades.
* Editar propiedades propias.
* Ver propiedades propias.

### TENANT_ADMIN

* Ver todas las propiedades del tenant.
* Administrar agentes.
* Compartir propiedades entre agentes.

### SUPER_ADMIN

* Acceso global.
